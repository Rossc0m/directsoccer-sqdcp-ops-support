💾 Data Management

📤 Export to CSV

Download all recorded SQDCP data as a CSV file — open in Excel for further analysis or share with the team.

📥 Import from CSV

Restore data from a previously exported CSV. Existing records for the same date are kept; only new dates are imported.

📋 Copy as JSON

Copy all data to clipboard as JSON — useful for moving data to another browser or device manually.

🗑️ Clear All Data

Permanently delete all SQDCP records from this browser. This cannot be undone — please export a CSV backup first.

ℹ️ How data is stored: Records are saved in your browser's local storage on this device. To access data across multiple devices, export a CSV from one device and import it on another — or nominate one device as your primary entry point and share exported reports.
